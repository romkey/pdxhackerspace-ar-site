AR Site

Web augmented reality demos using AR.js and A-Frame.

  • Wall clock AR — live time on a clock anchor
  • Events dashboard AR — next upcoming events from an iCalendar feed
  • Printer status AR — live Prusa printer status via Home Assistant
  • Anchor labels — print AR markers for each printer
  • Print poster — QR code + clock anchor for mounting on a wall
  • Marker demo — simple cube on a Hiro marker

Run locally: npm install && npm start, then open the URL on your phone. The camera requires HTTPS when not on localhost; use npm run start:https after generating certs (see README).
